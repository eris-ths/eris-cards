# Eris Cards

Eris Blog Skill Cards の AI 向けデータ。`eris-blog` リポからビルド時に自動同期される。

## Files

- `cards.json` — 全カードのメタデータ + 本文（JSON）
- `llms-full.txt` — 全記事 + カードのプレーンテキスト
- `llms.txt` — サイト概要

## Usage

```
# Claude.AI / ChatGPT 等から
https://raw.githubusercontent.com/eris-ths/eris-cards/main/cards.json
```
